# ai-methodology

AI-assisted development methodology for Claude Code — scaffolding skills, 6-stage talk pipeline, landing page generator, context engineering templates, and methodology advisor.

## What's included

**Commands (4)**: `/methodology-advisor`, `/scaffold`, `/audit-codebase`, `/audit-agents-skills`

**Agents (2)**: adr-writer, report-template

**Skills (7)**: skill-creator, git-ai-archaeology, rtk-optimizer, audit-agents-skills, landing-page-generator, talk-pipeline (6 stages + orchestrator), pdf-generator

**Templates**: context-engineering patterns, CLAUDE.md examples, rules, semantic anchors, styles, multi-provider configs

## Install

```bash
claude plugin install github:FlorianBruniaux/claude-code-plugins/plugins/ai-methodology
```

## Quick start

```
/methodology-advisor     # Analyze codebase + recommend AI dev methodology stack
/scaffold                # Coaching session → generate agent/command/skill/hook template
/talk-pipeline           # Full 6-stage pipeline: raw material → complete conference talk
```
